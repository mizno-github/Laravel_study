## プロジェクトの立ち上げ方

- gitからクローンする

    ```bash
    $ git clone 'URL'
    ```

- dockerを立ち上げる

    ```bash
    $ docker-compose exec app bash
    ```

- laravelの準備

    ```bash
    $ touch .env
    $ cat .env.example >> .env
    # envファイルは個人に合わせ修正する
    $ composer autoload
    $ php artisan key:generate
    $ php artisan migrate
    ```

- 下記URLにアクセスできれば環境構築完了です

    [localhost](http://127.0.0.1:8000)
